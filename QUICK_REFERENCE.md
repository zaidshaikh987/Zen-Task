# Quick Reference Card - ZenTasks Features

## 🎹 Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `N` | New task dialog |
| `F` | Focus Mode (Pomodoro) |
| `S` | AI Standup report |
| `?` | Show shortcuts panel |

## 🎙️ Voice Commands

### Landing Page
- "Go to Dashboard"
- "Go to Analytics"
- "Go to Settings"
- "Go to Home"

### Dashboard
- "Add high priority task: [task name]"
- "Add low priority task: [task name]"

## 🎯 Feature Locations

### Dashboard Header
- 🔥 **Burn Down Ticker** - Shows tasks remaining
- 🎤 **Voice Input** - Mic icon for voice task creation
- 🎯 **Focus Mode** - Button to enter Pomodoro mode
- 👥 **Standup** - AI standup report generator
- ➕ **New Task** - Create task dialog

### Kanban Board
- 🖱️ **Live Cursors** - 3 team members moving around
- ⚡ **Spring Physics** - Cards bounce when dragged
- 🗑️ **Trash Zone** - Appears at bottom during drag
- ✨ **AI Sort** - Button on each column (2+ tasks)
- ⚠️ **WIP Warning** - Doing column (>3 tasks)
- 🔴 **Overdue Decay** - Red glow on old tasks
- 🕒 **Timestamps** - "2h ago" on each card

### Analytics Page
- 📊 **Charts** - Bar chart, pie chart
- 📈 **Velocity Heatmap** - 12 weeks of completions
- 📉 **Stats Cards** - Total, In Progress, Completed, High Priority

### Landing Page
- 🎙️ **Voice Navigation** - Floating mic button (bottom-right)

## 🎨 Visual Indicators

### Priority
- 🔴 **High Priority** - Red badge, pulse animation
- ⚪ **Low Priority** - Gray badge

### Status Colors
- 🔵 **To Do** - Blue theme
- 🟡 **Doing** - Amber theme
- 🟢 **Done** - Emerald theme

### Urgency (Burn Down Ticker)
- 🟢 **Morning** (before 12pm) - Green
- 🟡 **Afternoon** (12pm-5pm) - Amber
- 🔴 **Evening** (after 5pm) - Red

### Overdue Decay
- ⏱️ **0-2 hours** - Normal
- 🟠 **2-24 hours** - Gradual red glow
- 🔴 **24+ hours** - Full red glow

## 🎬 Quick Demo Flow

1. **Landing** → Click mic → Say "Go to Dashboard"
2. **Dashboard** → Press `N` → Create task
3. **Voice Task** → Click mic → Say "Add high priority task: Demo feature"
4. **Drag Card** → Watch spring physics
5. **AI Sort** → Click "AI Sort" button
6. **Focus Mode** → Press `F` → Start Pomodoro
7. **Standup** → Press `S` → View report
8. **Trash** → Drag card to trash zone
9. **Analytics** → View velocity heatmap

## 🔧 Troubleshooting

### Voice not working?
- Use Chrome, Edge, or Safari
- Check microphone permissions
- Speak clearly and wait for listening indicator

### Features not showing?
- Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
- Clear localStorage: `localStorage.clear()` in console

### Performance issues?
- Close other browser tabs
- Reduce number of tasks
- Clear browser cache
